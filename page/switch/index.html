下面程序介绍
下载一个cisco的包ciscolib-0.1.tar.gz（思科的交换机）

tar -zxf ciscolib-0.1.tar.gz
cd ciscolib-0.1
python setup.py build
python setup.py install


cat switch.py
#!/usr/bin/env python
#conding:utf-8
import time
import ciscolib

IP='ip' #交换机地址
PASSWORD="cisco"
USERNAME=""
ENABLE_PW="cisco"

cmd = ['configure t','interface f0/1','shutdown','no shutdown']
try:
    sw = ciscolib.Device(IP,PASSWORD,ENABLE_PW)
    count = 0
    sw.connect()
    while (count < 1000):
        time.sleep(5)
        sw.enable(ENABLE_PW)
        sw.cmd(cmd[0])
        sw.cmd(cmd[1])
        sw.cmd(cmd[2])
        time.sleep(2)
        sw.cmd(cmd[3])
        count += 1
    sw.disconnect()
except Exception as e:
    print e
#注意超时时间，sleep是不是超过交换机超时时间,
这样实时性要求比较强的程序会有很大影响，如果没影响可以调节端口duwn的时间
设置一个定时任务，每天晚上十二点开始执行，大部分人这个时候都睡觉了，个别奇葩除外
这样就可以好好睡觉了
